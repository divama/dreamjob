## Setup development environment:
1. Make sure the following is installed on your machine:
    - [Node.js](http://nodejs.org/)
2. Run this command to install the global dependencies:
    ```
    npm install -g gulp
    ```
3. Clone repo from [https://github.com/divama/dreamjob.git](https://github.com/divama/dreamjob.git)
4. Run these commands to install the project-specific dependencies:
    ```
    npm install
    ```
    or to only run the project locally without project-specific dependencies for developing run
    ```
    npm install --only=production
    ```
5. You can now start your development server.
    ```
    npm start
    ```
6. Done! Enjoy!
